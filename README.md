# Temperature Monitoring System

Python project that simulates and analyzes temperature sensor readings.

## Features

- Manual temperature input in Celsius or Fahrenheit
- Automatic conversion to Celsius
- Simulate 10 sensor readings
- Detect spikes and trends
- Show average, maximum, and minimum values
- Save data to file
- Load saved data from file

## Skills Used

- Python
- Lists
- Loops
- Functions
- File handling
- Input validation
- Debugging
